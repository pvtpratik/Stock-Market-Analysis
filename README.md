# 📈 Stock Market Analysis – Interactive Data Analytics Dashboard

## 📌 Project Overview

The **Stock Market Analysis Dashboard** is an end-to-end **data analytics & business intelligence project** built using **SQL, Excel, Power BI, and Tableau**.

This project focuses on analyzing **stock performance, portfolio returns, volatility, and trading behavior** across multiple companies and sectors.

It provides actionable insights for **investors, analysts, and decision-makers** through interactive dashboards and KPI-driven analysis.

## 🎯 Business Objectives

* Analyze **stock market performance** across multiple companies

* Identify **top-performing sectors and companies**

## 🧮 Key Formulas Used

### 📌 Daily Return

```sql
Daily Return = (Previous Close - Current Close) / Current Close
```

### 📌 Volatility (Standard Deviation of Returns)

```text
σ = √( (1/N) * Σ (Ri - R̄)² )
```

### 📌 Sharpe Ratio

```text
Sharpe Ratio = (Average Return - Risk-Free Rate) / Volatility
```
