# 📈 Stock Market Analysis – Interactive Data Analytics Dashboard

## 📌 Project Overview

The **Stock Market Analysis Dashboard** is an end-to-end **data analytics & business intelligence project** built using **SQL, Excel, Power BI, and Tableau**. This repository contains:

* 📊 Interactive dashboards (Power BI & Tableau)
* 🗄️ SQL queries used for data preparation and KPI calculations
* 🖼️ Dashboard preview images for quick visualization

The project focuses on analyzing stock price trends, returns, volatility, and overall market performance across multiple companies and time periods.

---

## 🔗 Live Assets in This Repository

* 📁 **Power BI Dashboard (.pbix)** – Interactive slicers & measures  
* 📁 **Tableau Workbook (.twbx / .hyper)** – Advanced filtering & drill-down  
* 📁 **SQL Queries** – `Stock_Market_SQL.sql`  
* 🖼️ **Dashboard Preview Images** – Excel, Power BI & Tableau  

---

## 🎯 Business Objectives

* Analyze stock performance across multiple companies  
* Measure **Daily Returns** to track price changes  
* Evaluate market risk using **Volatility (Standard Deviation)**  
* Identify trends across **Year → Month → Day**  
* Compare performance across different stocks  

---
